# Sorted Squared Array

Given a sorted array containg integers. Return the array containing the squares of all the elements of the array in sorted order.

### Example 1:
```
Input:  1 2 3 5 9
Output: 1 4 9 25 81
```

### Example 2:
```
Input:  -10 -1 2 3 5 
Output: 1 4 9 25 100
```

### Constraints:
- array contains integer elements

### Complexities:
- O(N) Time
- O(N) Space
